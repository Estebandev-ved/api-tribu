# 🏹 Tribu - E-commerce de Próxima Generación

![Estado del Proyecto](https://img.shields.io/badge/Estado-En_Desarrollo-orange)
![Tecnologías](https://img.shields.io/badge/Stack-Java_%7C_Spring_Boot_%7C_React-blue)

**Tribu** es una plataforma de e-commerce diseñada para ofrecer una experiencia de compra fluida, segura y moderna. Más allá de un mercado convencional, Tribu está evolucionando hacia un ecosistema autónomo potenciado por Inteligencia Artificial.

---

## 🚀 Características Actuales

### 💻 Frontend (React + Vite)
- **Interfaz Premium**: Animaciones fluidas con `Framer Motion` y diseño intuitivo.
- **Billetera Virtual (Wallet)**: Gestión de saldo y métodos de pago integrados.
- **Panel de Administración**: Control total sobre productos, pedidos y usuarios.
- **Experiencia de Usuario**: Carrito reactivo, notificaciones en tiempo real y flujo de checkout optimizado.

### ⚙️ Backend (Spring Boot)
- **Arquitectura Robusta**: Construido con Java 21 y arquitectura limpia.
- **Seguridad Avanzada**: Autenticación y autorización mediante Spring Security y JWT.
- **Gestión de Datos**: Integración con MySQL y JPA para una persistencia eficiente.
- **Servicios Automatizados**: Sistema de correos electrónicos y validaciones de negocio rigurosas.

## 🔮 Visión de Futuro: Tribu AI Agency

Nuestra meta es transformar Tribu de un e-commerce tradicional a una **Agencia de Agentes de IA**. Estos agentes no serán simples bots, sino trabajadores autónomos que operarán la tienda como una agencia profesional:

- **🤖 Agente de Inventario**: Predicción de stock y pedidos automáticos a proveedores basados en tendencias.
- **🎨 Agente Creativo**: Generación automática de descripciones de productos, banners y contenido visual de alta calidad.
- **📈 Agente de Marketing**: Ejecución de campañas personalizadas y optimización de precios en tiempo real.
- **💬 Agente de Soporte Concierge**: Atención al cliente hiper-personalizada que resuelve problemas de extremo a extremo.
- **🔍 Agente de Análisis**: Dashboards predictivos que informan sobre el próximo gran movimiento del mercado.

*Tribu no es solo un sitio de compras; es el futuro del comercio autónomo.*

## 📥 Instalación y Configuración

### Requisitos Previos
- JDK 21
- Node.js (v18+)
- MySQL Server

### Backend
```bash
# Navegar al directorio raíz
./mvnw spring-boot:run
```

### Frontend
```bash
cd tribu-frontend
npm install
npm run dev
```
